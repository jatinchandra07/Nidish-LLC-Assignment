# 🚀 Full Stack Content Builder

A production-ready full-stack application that demonstrates form handling, validation, API communication, and dynamic UI updates.

---

## 📌 Overview

This application allows users to create and preview content dynamically. Users can input a heading, paragraph, background image URL, and text color, and instantly see a live preview. The data is validated on both frontend and backend before being processed.

---

## ✨ Features

- 🧾 Two-column layout (Form + Live Preview)
- ⚡ Real-time preview updates
- ✅ Client-side validation
- 🔒 Server-side validation
- 🌐 REST API integration
- 🎨 Dynamic UI styling (background image & text color)
- 🔔 Toast notifications for user feedback
- 🧱 Scalable and modular architecture

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios
- React Toastify

### Backend
- Node.js
- Express.js
